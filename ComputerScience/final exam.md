## 問答題
- Von Neumann architecture
  - 以運算單元為中心
  - 採用儲存程序原理
  - 記憶體是按位址存取、線性編址的空間
  - 控制流由指令流產生
  - 指令由操作碼和位址碼組成
- instruction cycle
  - 擷取 : 將指令存入指令寄存器(IR)
  - 解碼 : 在IR裡的指令轉換為機器語言
  - 執行 : 解碼後的信息以控制信號的傳遞給CPU
  - 儲存結果
- HTML
  - HyperText Markup Language
  - 使用(tag)標籤和指令碼
  - 用於設計網頁
  - 用來結構化資訊
  - 可以嵌入CSS&JS
- CSS
  - Cascading Style Sheets
  - 修飾網頁
  - 靜態的
  - 擁有對網頁物件和模型樣式編輯的能力
  - 被大部分瀏覽器支援
- Javascript
  - 支援物件導向程式設計
  - 不支援I/O
  - 指令式編程
  - 函式語言程式設計
  - 動態化
- RWD
  - Responsive Web Design
  - 可使網站在不同的裝置上瀏覽時對應不同解析度皆有適合的呈現
